# 🎮 PS HUB

PS HUB is a Game Library CRUD web application developed using **Node.js**, **Express.js**, **EJS**, **Bootstrap**, and **JavaScript**. It provides separate roles for **Admin** and **User**, allowing admins to manage games while users can browse, purchase, and track their orders.

## 🚀 Live Demo

🔗 **Live Project:** https://ps-hub-1.onrender.com

---

## 📌 Features

### 👨‍💼 Admin
- Admin Login
- Add New Games
- Edit Game Details
- Delete Games
- View All Orders
- Mark Orders as Delivered

### 👤 User
- User Registration & Login
- Browse Available Games
- Buy Games
- View Order History
- Secure Session Management

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Santhoshs0106/PS-HUB.git
```

Navigate to the project:

```bash
cd PS-HUB/ps-hub-crud
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open your browser:

```text
http://localhost:3000
```
